# 📱 Transactions App

A modern iOS Transactions app built with **clean architecture**, **modular Swift packages**, and a focus on **scalability, testability, and performance**.

---

## ✨ Features

- 📄 View a list of transactions  
- 🔍 Detailed transaction view  
- 💳 Support for credit and debit transactions  
- ⚡ Fast loading with local caching  
- 🔄 Incremental refresh using `since` strategy  
- 🎨 Clean, reusable SwiftUI components
